# Online-Calculator Test Script

Tests the addition, subtraction, and division functionalities of the online calculator

## Installation
```bash
npm install
```

## Execution

The test case can be executed with the following command:
```bash
npm test
```

## Tools Used

* Cucumber 7.0.0
* Chai 4.2.0,
* Selenium Webdriver 4.0.0 
* Sleep 6.3.0
* Tesseract.js 2.1.4
